# OSA — Context

## Project Type
Single-page academic paper presentation (CVPR 2026). Static Astro site deployed to GitHub Pages subfolder `/osa/`.

## Domain Terms

| Term | Definition |
|------|-----------|
| OSA | Orthogonalized State Attention — the proposed method |
| OSU | Orthogonalized State Update — mechanism that constrains state evolution on Stiefel manifold to prevent rank collapse |
| APFE | Anatomical Prior-aware Feature Enhancement — module that decouples anatomical structures from speckle noise via physics-driven decomposition |
| CAMUS | Cardiac Ultrasound dataset (public benchmark) |
| EchoNet-Dynamic | Stanford echocardiography video dataset |
| mDice | Mean Dice coefficient — segmentation accuracy metric |
| Rank Collapse | Progressive singular value decay in state matrix under severe speckle noise, causing anatomical details to be overwhelmed |

## Routing

- `/osa/` → `src/pages/index.astro` (meta refresh redirect to `/osa/en/`)
- `/osa/en/` and `/osa/zh/` → `src/pages/[lang]/index.astro` → `HomePage.astro`
- `/osa/en/poster`, `/osa/zh/poster` → `src/pages/[lang]/poster.astro` (fullscreen iframe)
- `/osa/en/slides`, `/osa/zh/slides` → `src/pages/[lang]/slides.astro` (fullscreen iframe + zoom/print controls)

## Assets

Publication figures and logos are served from the centralized `mykcs/academic` repository at `wangrui2025.github.io/academic/`. Paths use `/academic/images/...` (see ADR-0002).

## i18n

Content Collections under `src/content/homepage/` (en.json, zh.json), via `t(lang, key)` function in `src/i18n/index.ts`.
